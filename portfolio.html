Web Development Portfolio

OleOle

OleOle is a sports gaming web app for viewing stats and betting on Major League Soccer games. The front end was built with Bootstrap and Charts.js and we used mySQL, sequelize, and express.js on the back end.

OrlandoEats

OrlandoEats is a web app for finding and saving dining options in the Orlando area. We built the front end with Materialize and used the Zomato and Google Maps APIs to return information to the user.

GitHub Repo Project Site
Friend Finder

Friend Finder is a friend pairing app, where users answer a few questions and are then matched with a new best friend based on their answers. I built this app with node.js, express.js, and mySQL

GitHub Repo Project Site
Bamazon

Bamazon is command line based store front similar to amazon. It has two user interfaces. In the customer interface, you can buy products from the store. In the manager interface, you can add inventory and new products. I built this app using node.js and mySQL.

GitHub Repo
Crystal Collector

Crystal Collector is a game where you click on crystals that appear on the screen. As you click crystals their point value is added to your score. The goal is to get your score to match the goal score given to you. Built with HTML, css, and javascript.

GitHub Repo Project Site
8-Bit ToDos

8-Bit ToDos is a to do list web app themed after classic 8-bit arcade games. It allows the user to add tasks to the list and also check them as complete. I built this app with express.js, handlebars, and mySQL.
